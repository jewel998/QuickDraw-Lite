QuickDraw-Lite

This is an implementation of Google's famous online game project QuickDraw,a game which recognize drawings.This page is implemented using tensorflow.js and the model is trained using keras framework. This implemenatation is able to recognize 100 classes of drawings as the Quick, Draw dataset is the largest image dataset with a total of 345 classes of drawing. It achieves 6% top 5 error. The accuracy could be improved but I focused on making the model light with around 1 million parameters. The table will show the top 5 predictions.
